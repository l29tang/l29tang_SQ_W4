# Getting Over It — Choose Your Own Adventure
Task was to create 15 files and make interactive multiple choice scenes. 
## What This Example Demonstrates

A choose-your-own-adventure game built with p5.js, themed around 
*Getting Over It with Bennett Foddy*. The game demonstrates how to 
split a multi-screen interactive story across multiple JavaScript files, 
manage game state, draw visuals purely with code (no images), and handle 
mouse click interactions to navigate between scenes.

## Setup and Interaction Instructions

To run the sketch locally, open `index.html` in Google Chrome using Live Server.

Click **Begin the Climb** to start. Read each scene and click one of the 
two buttons to make your choice. Your decisions branch the story across 
four levels, leading to one of eight possible endings. Click **Try Again** 
or **Play Again** to restart from the beginning.

**Opening the Chrome Console**

- **Windows:** Press `F12` or `Ctrl + Shift + J`, then click the **Console** tab
- **Mac:** Press `Cmd + Option + J`

The console will show any errors in your sketch.

## Assets

No external assets used. All visuals are generated with p5.js using 
shapes, text, and colour — no images or sounds.
